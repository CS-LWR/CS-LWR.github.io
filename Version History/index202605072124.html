About Me

A concise introduction to my academic background, research direction, and technical interests.

I am interested in signal processing, electromagnetic forward and inverse modeling, magnetic induction sensing, and artificial intelligence for scientific computing. My work focuses on building models that are not only computationally useful, but also physically interpretable.

My recent research direction is frequency-domain electromagnetic modeling for underground sensing. In particular, I am developing simplified yet meaningful inversion structures that estimate conductivity distributions from multi-frequency transmitter-receiver coil responses.

Research Focus

My current work aims to connect physically rigorous electromagnetic modeling with practical, explainable inversion strategies.

My current magnetic induction inversion method uses four sector-wise simplified 1D inversions. The underground region is divided horizontally according to transmitter-receiver geometry and vertically into ten fixed, evenly distributed depth intervals. The inversion estimates conductivity for each sector-depth cell, while magnetic permeability and permittivity are treated as known parameters.

Core Methodological Direction

  • Use a strict 3D forward model to generate physically meaningful electromagnetic responses.
  • Use a simplified sector-wise layered inversion model to keep the inverse problem interpretable.
  • Estimate conductivity distribution using multi-frequency magnetic induction measurements.
  • Analyze observability, parameter cross-talk, and sensitivity before moving toward higher-dimensional inversion.

Selected Projects

Magnetic Induction Layered Inversion

This project studies frequency-domain magnetic induction sensing for underground layered media. It focuses on conductivity inversion using transmitter-receiver coil responses and physically meaningful constraints.

  • Designed a sector-wise layered inversion strategy with fixed depth intervals.
  • Modeled multi-frequency secondary-field responses from coil measurements.
  • Analyzed observability and parameter cross-talk using Jacobian-based diagnostics.
FDEM Magnetic Induction Layered Media Inverse Problem

3D Electromagnetic Forward Modeling

This project explores a stricter 3D forward-modeling route for electromagnetic sensing scenarios. The model considers transmitter and receiver geometry, underground discretization, and spatially varying electromagnetic material properties.

  • Formulated a frequency-domain electromagnetic forward workflow.
  • Considered practical coil layout, receiver flux calculation, and boundary conditions.
  • Built a foundation for validating simplified inversion models using stricter simulation data.
3D Modeling Voxel Grid Forward Model Simulation

EEG Artifact Removal Using Deep Learning

This project investigates deep-learning-based EEG denoising methods, including convolutional autoencoders and variational autoencoders for artifact removal and signal reconstruction.

  • Implemented CNN-AE and VAE-style reconstruction models for EEG denoising.
  • Evaluated performance using RRMSE, PSD-RRMSE, and Pearson correlation coefficient.
  • Explored deployment-oriented workflows including segmentation, normalization, and inference.
