# Images Folder

Place your website images in this folder.

Recommended filenames already referenced by the website:

```text
profile.jpg
research-mi.jpg
research-eeg.jpg
research-wireless.jpg
project-mi-inversion.jpg
project-forward.jpg
project-eeg.jpg
project-website.jpg
project-communication.jpg
```

You can use `.jpg`, `.png`, or `.webp`, but if you change the filename or extension, also update the corresponding path in:

```text
data/site-data.js
```

Recommended image sizes:

- Profile image: 900 × 1100 px or similar portrait ratio
- Project images: 1200 × 800 px
- Research images: 1200 × 800 px

Keep images compressed so the website loads quickly.
